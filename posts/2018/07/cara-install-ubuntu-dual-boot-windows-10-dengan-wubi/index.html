Cara Menginstal Ubuntu Menggunakan WUBI[1]

Klik dua kali pada WUBI yang di-download dan klik “Ya” ketika ditanya apakah Anda ingin menjalankannya melalui keamanan Windows.

Sebuah jendela akan muncul dan akan terlihat seperti image terlampir.

Untuk menginstal Ubuntu:

  • Pilih drive Anda ingin menginstal Ubuntu ke dengan memilih drive yang sesuai di bawah “Instalasi Drive”.
  • Pilih berapa banyak ruang disk untuk disisihkan untuk Ubuntu dengan memilih ukuran di bawah “Ukuran Instalasi”.
  • Pilih lingkungan desktop dari daftar drop down.Â Daftar ini sedikit membingungkan.Â Untuk Ubuntu default dengan tongkat desktop Unity dengan “Ubuntu”.Â “Ubuntu GNOME” memiliki lingkungan desktop GNOME yang sangat mirip dengan Unity.Â Kubuntu menggunakan lingkungan desktop Plasma KDE, Xubuntu menggunakan lingkungan desktop XFCE yang ringan dan sangat dapat dikustomisasi, Lubuntu menggunakan lingkungan desktop LXDE yang ringan.Â Ada berbagai pilihan lain yang kurang dikenal.
  • Pilih bahasa yang akan digunakan untuk Ubuntu dengan memilihnya dari drop down.
  • Masukkan nama pengguna default untuk digunakan dengan Ubuntu
  • Masukkan kata sandi dan ulangi di kotak di bawahnya
  • Klik “Pasang”

Penginstal WUBI sekarang akan mengunduh versi Ubuntu yang terkait dengan versi WUBI yang Anda unduh dan kemudian akan menciptakan ruang untuk menginstalnya.

Anda akan diminta untuk reboot dan ketika Anda melakukan Ubuntu akan memuat dan file-file akan disalin dan diinstal.

Bikin dekstop shortcut untuk boot langsung ke OS lain dalam dualboot / multiboot

Kalau agan2 user dualboot atau multiboot dan menggunakan bootloader windows sebagai bootloader default, mungkin trik ini bisa berguna. Oiya, disi ane pake bios legacy sama tipe partisi mbr, ngga tau deh buat uefi-gpt.

Awalnya sih ane ngga kepikiran, ceritanya pas ane instalÂ phoenixÂ os pake installer exe malah dibikinin otomatis desktop shorcutnya. Kebetulan sebelumnya ane udah dualboot sama Ubuntu, jadi kepikiran deh buat dekstop shortcut juga buat boot ke ubuntunya. Fungsinya ? Ya biar bisa langsung boot ke os sebelah tanpa harus nunggu booting trus pilih OS-nya manual.

Oke, langsung aja, pertama copy script ini ke notepad :

C:\Windows\System32\bcdedit /bootsequence {d3e10d24-5db9-11e8-82aa-dc85de38885b}
shutdown /r /t 0

*Ganti angka-angka dalam kurung kurawal sesuai dengan kode identifier / resume objectnya.
Agan2 bisa mengeceknya dengan membuka cmd sebagai admin, ketikkan bcdedit, tekan enter, akan keluar informasi mengenai bootlistnya, atau bisa juga dilihat lewat tool EasyBCD (kalau ada)

Simpan file tadi dengan ekstensi .bat, bikin deh shorcut file .bat tadi ke dekstop, ganti iconnya biar lebih mantep.

Cuman kurangnya disini, perintah .bat itu harus dieksekusi dgn run as admin, jadi harus klik kanan dulu, trus pilih run as admin, repot kan.

Nah, biar ngga repot terus2an, kita bisa bikin elevated shortcut buat file .bat nya aja, tutornya disini :

https://winaero.com/blog/create-elevated-shortcut-to-skip-uac-prompt-in-windows-10/

Biar praktis pake tool winaero tweaker aja buat bikin shortcutnya.

Ada alternatif lain sih yang lebih gampang, pake aplikasi iReboot, EasyBCD emang cocok mah kalo diduetin sama iReboot!

http://neosmart.net/iReboot/
