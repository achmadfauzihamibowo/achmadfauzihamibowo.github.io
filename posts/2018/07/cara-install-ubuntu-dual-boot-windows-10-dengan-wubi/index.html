Cara install Ubuntu Dual Boot Windows 10 dengan Wubi

Baik, kali ini kita akan mencoba menginstall Ubuntu dengan Wubi, kalau ada yang belum tahu Wubi itu apa, Wubi adalah salah satu tools untuk menginstall Ubuntu dengan mudah melalui Windows tanpa harus melakukan repartisi, cocok untuk pemula saat bermain Ubuntu Linux.

Wubi sendiri adalah akronim dari Windows-based Ubuntu Installer jadi ini memang tools yang diperuntukan untuk pengguna Windows! Sebenarnya Wubi ini sendiri sudah menjadi tools yang deprecated dari Ubuntu secara resmi, tetapi ada seorang pengembang di GitHub yang mengembangan kembali tools ini agar bisa dipergunakan untuk menginstall Ubuntu rilis terbaru (berserta flavor atau turunannya)

Tampilan memilih OS saat Dual Booting berjalan setelah menginstall Wubi.

Cara mendapatkan Wubi terbaru?

Untuk mendapatkan rilis Wubi terbaru, bisa melalui rilis langsung dari GitHub berikut ini:

https://github.com/hakuna-m/wubiuefi/releases

Halaman yang ditautkan memiliki sejumlah versi yang berbeda. Rilis LTS terbaru adalah 18.04 jadi jika Anda menginginkan versi yang didukung penuh untuk beberapa tahun ke depan, temukan tautan unduhan untuk 18.04. Ini adalah tautan tertinggi pada halaman ini.

Sesuaikan dengan versi Ubuntu yang kalian unduh ya!

Versi terbaru Ubuntu saat artikel ini ditulis adalah versi 18.04 LTS dengan codename Bionic Beaver

Untuk yang belum memiliki Ubuntu terbaru, bisa mengunduhnya melalui beberapa situs dibawah ini:

Unduh** disini!**

Cara Menginstal Ubuntu Menggunakan WUBI[1]

Klik dua kali pada WUBI yang di-download dan klik “Ya” ketika ditanya apakah Anda ingin menjalankannya melalui keamanan Windows.

Sebuah jendela akan muncul dan akan terlihat seperti image terlampir.

Untuk menginstal Ubuntu:

  • Pilih drive Anda ingin menginstal Ubuntu ke dengan memilih drive yang sesuai di bawah “Instalasi Drive”.
  • Pilih berapa banyak ruang disk untuk disisihkan untuk Ubuntu dengan memilih ukuran di bawah “Ukuran Instalasi”.
  • Pilih lingkungan desktop dari daftar drop down. Daftar ini sedikit membingungkan. Untuk Ubuntu default dengan tongkat desktop Unity dengan “Ubuntu”. “Ubuntu GNOME” memiliki lingkungan desktop GNOME yang sangat mirip dengan Unity. Kubuntu menggunakan lingkungan desktop Plasma KDE, Xubuntu menggunakan lingkungan desktop XFCE yang ringan dan sangat dapat dikustomisasi, Lubuntu menggunakan lingkungan desktop LXDE yang ringan. Ada berbagai pilihan lain yang kurang dikenal.
  • Pilih bahasa yang akan digunakan untuk Ubuntu dengan memilihnya dari drop down.
  • Masukkan nama pengguna default untuk digunakan dengan Ubuntu
  • Masukkan kata sandi dan ulangi di kotak di bawahnya
  • Klik “Pasang”

Penginstal WUBI sekarang akan mengunduh versi Ubuntu yang terkait dengan versi WUBI yang Anda unduh dan kemudian akan menciptakan ruang untuk menginstalnya.

Anda akan diminta untuk reboot dan ketika Anda melakukan Ubuntu akan memuat dan file-file akan disalin dan diinstal.

Tutorial Bergambar

Tahap** pertama**, pindahkan file installer Wubi kedalam satu folder dengan file *.iso Ubuntu. Jalankan Wubi sebagai****_ Administrator (Run as Administrator)_

Tahap kedua yaitu konfigurasi instalasi Wubi, disini kita bisa memilih tempat dimana Ubuntu akan diinstall, yaitu di **Installation Drive (disarankan tidak ditaruh dengan OS Windows yaitu C:)**lalu ada Installation size nah ini adalah alokasi ukuran system yang akan dipergunakan untuk Ubuntu, maksimal mencapai 64 GB (sesuaikan dengan ukuran Harddisk kalian ya!) selebihnya kalian paham kan? Dan setelah mengatur Username dan Password, silakan klik Install.

Tahap** ketiga: **Instalasi Ubuntu akan berlangsung, silakan tunggu karena membuat file system itu cukup lama (dalam tutorial ini saya membuat file systemnya 25 GB dari maksimal 64 GB)

Selesai, kalian bisa langsung mencoba untuk reboot ke Ubuntu!

Pencopotan**/Uninstall Instalasi**

Bagaimana** cara mencopot/uninstall Wubi?**

Kalian dapat mencopot/menguninstall Wubi dengan cara

Cara pertama

Jalankan uninstaller di “Control Panel > Programs and Features” untuk Windows 7, Windows 8.1 atau Windows 10.

Cara kedua

Atau, Anda dapat menjalankan: C:\ubuntu\uninstall-wubi (disesuaikan dengan lokasi kalian menginstall Wubi ya!)

Bikin dekstop shortcut untuk boot langsung ke OS lain dalam dualboot / multiboot

Kalau agan2 user dualboot atau multiboot dan menggunakan bootloader windows sebagai bootloader default, mungkin trik ini bisa berguna. Oiya, disi ane pake bios legacy sama tipe partisi mbr, ngga tau deh buat uefi-gpt.

Awalnya sih ane ngga kepikiran, ceritanya pas ane instal phoenix os pake installer exe malah dibikinin otomatis desktop shorcutnya. Kebetulan sebelumnya ane udah dualboot sama Ubuntu, jadi kepikiran deh buat dekstop shortcut juga buat boot ke ubuntunya. Fungsinya ? Ya biar bisa langsung boot ke os sebelah tanpa harus nunggu booting trus pilih OS-nya manual.

Oke, langsung aja, pertama copy script ini ke notepad :

C:\Windows\System32\bcdedit /bootsequence {d3e10d24-5db9-11e8-82aa-dc85de38885b}
shutdown /r /t 0

*Ganti angka-angka dalam kurung kurawal sesuai dengan kode identifier / resume objectnya.
Agan2 bisa mengeceknya dengan membuka cmd sebagai admin, ketikkan bcdedit, tekan enter, akan keluar informasi mengenai bootlistnya, atau bisa juga dilihat lewat tool EasyBCD (kalau ada)

Simpan file tadi dengan ekstensi .bat, bikin deh shorcut file .bat tadi ke dekstop, ganti iconnya biar lebih mantep.

Cuman kurangnya disini, perintah .bat itu harus dieksekusi dgn run as admin, jadi harus klik kanan dulu, trus pilih run as admin, repot kan.

Nah, biar ngga repot terus2an, kita bisa bikin elevated shortcut buat file .bat nya aja, tutornya disini :

https://winaero.com/blog/create-elevated-shortcut-to-skip-uac-prompt-in-windows-10/

Biar praktis pake tool winaero tweaker aja buat bikin shortcutnya.

Ada alternatif lain sih yang lebih gampang, pake aplikasi iReboot, EasyBCD emang cocok mah kalo diduetin sama iReboot!

http://neosmart.net/iReboot/

Selesai!
