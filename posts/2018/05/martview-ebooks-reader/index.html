CARA INSTALASI

Catatan! Aplikasi ini bisa dijalankan di Windows 8 keatas dengan sistem 64 Bit dengan cara khusus, bagi pengguna sistem 64 Bit, silakan ikuti tutorial lanjutan.

Instalasi di Windows 7 keatas 32 Bit (x86)

  1. Unduh Java JRE versi terbaru, link download? ada di paling akhir post, download khusus yang x86 ya! (32 Bit)
  2. Install Java JRE terlebih dahulu sebelum menginstall MartView, lakukan penginstalan seperti biasa.
  3. Setelah Java JRE terinstall, lanjutkan dengan menjalankan installer MartView, next next saja ya!
  4. Selesai! MartView terinstall, silakan buka PDF kalian!

Instalasi lanjutan untuk pengguna 64 Bit (x64)

  1. Untuk pengguna sistem 64 Bit, cara installasi hampir sama dengan cara diatas, perbedaannya hanya pada Java JRE yang harus di install yaitu versi x86 (32 Bit) dengan x64 (64 Bit)
  2. Catatan untuk versi 64 Bit adalah dua versi Java JRE harus diinstall agar MartView dapat berjalan.
  3. Selebihnya, cara instalasi adalah sama dengan versi 32 Bit (x86)
