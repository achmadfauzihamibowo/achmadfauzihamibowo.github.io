Cara Membuat Installer Repack

Assallamualaikum.wr.wb

Hai guys, ga kerasa udah lama banget gue ga post, kali ini gue mau post random aja, ya kali ini tutorial yang mau gue bahas. Nah kali ini gue mau bahas cara bikin installer game yang biasa kalian download di internet, mungkin kalian udah paham betul sama nama CorePack BlackBox FitGirl Reloaded kan? pasti tau lah kalian ga mungkin nggak hehe. Nah sekarang kita mulai Cara Membuat Installer Repack Seperti CorePack

Oke disini gue mau jelasin tentang cara nge-repack game. Kalau yang dari tadi ga ngerti repack itu apa, oke gue jelasin. Jadi Repack itu adalah istilah game yang di compress secara penuh tanpa pengurangan fitur-fitur yang ada dalam game ataupun bila dikurangi hanya akan dikurangi pada segi Video ( Cut Scene ) dan pengurangan kualitas video tanpa mempengaruhi jalannya game tersebut, versi ini masih menjadi kesukaan di kalangan gamer yang menginginkan game dengan kualitas bagus. Tapi hanya 2 kekurangan dari Repack itu (Pada saat Menginstall Game Repack).

Pertama, Menginstall game-nya lama banget… Itu sudah jelas sekali karena cara menginstall game Repack itu sama seperti kita mengekstrak suatu file dengan menggunakan software seperti Win-Rar, 7 Zip, Win-Zip, Free Arc, dan Software Sejenisnya.. Karena dalam Nge-Repack game itu juga sama dengan kita mengompress suatu file. Padahal gamenya itu cuman berukuran 1 Gb pas ke Kompress akan terasa lambat, itu dikarenakan software itu akan mengkompress secara penuh untuk mendapatkan hasil yang terbaik…

Kedua, Kadang pas menginstall terjadinya Error, Seperti, : ISdone.dll Error Code-1, Archive Corrupt. Hal ini dikarenakan pada saat kita mendownload terjadi Error, yah,, mungkin itu karena koneksi yang sering putus nyambung,,,

Oke, mari membahas tentang Cara Membuat Installer Repack Seperti CorePack
Software yang perlu kita siapkan untuk membuat sebuah Repack game, yaitu adalah :

1. Inno Setup Enhanced Version = Software ini berguna untuk membuat File Installernya yang berbasis *.exe kayak ya setup.exe yang biasa kita lihat, dan Enhanced Version ini sangat bagus karena bisa membaca dan menerima Kode Script yang Tidak Valid, di Software Inno Setup biasa.

2. IMGBurn = Software ini berguna untuk membuat File ISO dengan cara gampang sekali, dan berguna untuk membuat file ISO yang kalian buat game Installer nya (kalau mau dibikin *.iso)

3. IcoFX = Software ini berguna untuk membuat File ICON, untuk memberi ICON di File Installernya atau Membuat ICON untuk Folder.

4. PhotoShop = Nggak perlu dijelasin lah ya, software ini nanti berguna untuk edit gambar di installer kita supaya lebih rapih juga.

5. WPI (Windows Phone Installer) Script by Razor12911 = Script dari Innosetup ini yang akan membuat installer seperti yang biasa CorePack pakai. Yang gue upload itu yang udah gue custom buat installer The Sims™ 4 hehe, buat contoh juga.

Biar kalian ga penasaran, ini gue kasih screenshot installer WPI versi gue nya.

Gimana? Mau bikin kan? Oke kita mulai!

(Tutorial ini tanpa video, gue males upload sekalian yang ga punya kuota banyak bisa cek artikel gue ini)

PART 1 | COMPRESS DATA

Compress data kalian dengan tools compressor, sebelum ngecompress data, kalian bagi-bagi file yang kalian mau bikin ke setup-1 nya, biasa gue bagi sampe 3,9 gb biar ukuran dvd, cut and paste data game nya ke 1 folder kayak gini nih.

datanya dibagi gitu, main cut paste ke folder kayak gini

Nah, setelah kalian bagi per file nya di satu folder, kita akan compress dengan tools Compressor yang udah gue upload, link paling bawah artikel semua ya, lu baca aja dulu wkwk. Cara pakenya gampang kok kayak gini nih.

Pertama kita mau bikin setup-1 untuk file pertamanya, extension bisa terserah lu gue aja bikin setup-1.lol nah gini maksud gue.

Nah kalau ga mudeng nih gue contohin yang udah gue masukkin data nya

Nah udah paham kan kenapa gue bilang bikin ke satu folder, itu maksud gue biar gampang juga bikin part setup nya. setelah itu klik compress bakal muncul cmd kayak gini nanti, tapi ga ampe finish ya gue males nungguinnya wkwk. Yang sabar ya nungguinnya tergantung cara compress sama berapa gede file nya.

Nah ntar kalau udah selesai ini, silahkan lu bikin lagi file setup- selanjutnya sesuai sama file yang udah lu part ke satu folder yang lain. nah ntar kalo udah jadi semua lu bakal punya file part, inget penamaan file partnya yak, setup-1 untuk data 1, setup-2 untuk selanjutnya (nama opsional intinya lu ngerti maksud gue) contoh gini

Nah yang gambar di kiri itu udah jadi setup-1 dst, yang kanan itu folder data yang di part untuk di compress, ngerti kan?

Oke gue rasa tahap Compress kalian udah ngerti, sekarang mulai bikin Installer nya.

PART 2 | BUAT INSTALLER
Pertama, kalian harus install Inno Setup Enhanced Edition yang bakal gue share, ini khusus buat bikin installer yang lebih lengkap dari InnoSetup biasa, setelah di install kalian juga harus udah download WPI Script by Razor yang nanti bisa kalian download semua di link diakhir post gue.

Setelah install inno setup, kalian load script WindowsPhone.iss dari file WPI yang udah kalian download dan kalian ekstrak, kalau bingung gini maksudnya.

Saat pertama buka InnoSetup, pilih More files… seperti diatas.

Pilih WindowsPhone.iss nah lalu kebuka lah script kayak dibawah.

Oke sekarang gue jelasin kode script disini ya!

[Installer]
Name=The Sims 4 (Isi sesuai nama game nya)
Publisher=Electronic Arts (Isi publishernya)
Version=1.0.0.0 (Isi sesuai versi game nya)
Size=15781 (Ini konvert ke MegaByte ya, ini artinya 15 GB)
;Files=C:\Program Files\Electronic Arts\The Sims 4 (pilih nanti game bakal keinstall dimana)
Watermark=SickPerson Games Store (Kalian bisa nambahin nama kalian di watermark installer)
Autorun=1

[Theme]
;Interface either be in dark or light
;Colors are Originally from OS Windows Phone 8 (01-20)
Image=1
Dark=1  
Color=2 (pilih sendiri, silahkan di coba 1-20 yang sesuai)

[Execs]
;Tile Caption|Exec Directory|Exec File|Shortcut Name/Action Name
AppExe1=The Sims™ 4|Game\Bin|TS4.exe|The Sims™ 4 (tempat dimana lu bikin shortcut dan segala macem, ikutin petunjuknya)
AppExe2=The Sims™ 4 (64-Bit)|Game\Bin|TS4_x64.exe|The Sims™ 4 (64-Bit) (sama ini juga khusus exe yang 64 bit di game)

Kalau ga ada exe 64 bit atau mau disable kayak gini
AppExe2=The Sims™ 4 (64-Bit)|Game\Bin|TS4_x64.exe|The Sims™ 4 (64-Bit)

ditambah ; di depannya jadi. dan ini berlaku di semua script ini.

;AppExe2=The Sims™ 4 (64-Bit)|Game\Bin|TS4_x64.exe|The Sims™ 4 (64-Bit)

[Redists]
;Redist Caption|Exec Directory + File|Arguments
Redist1=Direct X 9.0c|Redist\DirectX\DXSetup.exe|/silent (silahkan kalian pelajari contoh gue ini)
Redist2=VCRedist|Redist\VCRedist\VC.RedistInstaller1.6.exe|/silent

[Archives]
;Precomp 0.38, 0.40, 0.41 and 0.42
;Archive Types 7Zip, Freearc and WinRAR
PrecompVer=0.38
Archive1Type=Freearc
Archive1Source={src}\setup-1.lol (kalian sesuai in sama nama file nya)
Archive1Output={app}
Archive1Disk=1 (lu mau taruh di disk 1 silahkan tulis disk 1)
Archive1Password=sayangfika (sesuai sama password pas dibikin, kalo gapake kosongin aja)
;Precomp 0.38, 0.40, 0.41 and 0.42

Kalau file nya ada banyak dan ada berapa setup-* gimana?
Gini caranya, copy script atas dibawah tag [Archives] kayak gini (dan seterusnya)

BIAR GA SALAH, SESUAIIN SAMA FILE NYA, KALO KALIAN BIKIN UNTUK ARCHIVE 2 GANTI SEMUA YANG Archive1 KE Archive2 DAN SETERUSNYA, KAYAK GINI.

;Archive Types 7Zip, Freearc and WinRAR
PrecompVer=0.38
Archive2Type=Freearc
Archive2Source={src}\setup-2.lol
Archive2Output={app}
Archive2Disk=2
Archive2Password=sayangfika

[Registry]

Untuk registry khusus yang export file *.reg buka notepad trus copy di bawah tag [Registry]
Oke gue rasa masalah script installer udah ngerti fungsinya, nah silahkan kalian coba, untuk compile script atau ubah script SELALU SAVE SCRIPT TERLEBIH DAHULU BARU DI COBA. Karena kalau ga disave script yang di compile make script lama nya.

Kalau compile klik Ctrl+F9 kalau mau test script setelah save klik aja F9.
PART 2 | GRAFIS INSTALLER
Nah sekarang tinggal perbagus installer kalian dengan gambar game nya, silahkan cari di google gambar gamenya kayak gini

Nah kalian download yang banyak tapi ga banyak banget yang menurut kalian bagus, setelah itu buka folder WPI yang ada script nya, buka folder Graphics nah disitu ada 2 folder yaitu Images sama Tiles

Nah kalian buka kedua folder itu, bakal nemu banyak gambar, silahkan kalian sesuaiin sama pattern yang ada, tinggal seret gambar baru ke gambar lama lalu sesuaiin.

Kalian bisa pake photoshop buat edit gambar di Images sama Tiles, gue ga bahas cara editnya, bisa kalian googling wkwkwk, nah kalau udah selesai edit gambar balik lagi ke script WindowsPhone.iss yang ada di InnoSetup, sebelum compile kalian bisa ganti file icon yang ada di folder WPI dengan nama Icon.ico dengan icon kalian.

Setelah kalian rasa udah beres semua, kalian bisa compile script nya dengan cara yang gue bilang diatas, atau dengan penjelasannya.

Di ribbon ada Build, klik Build => Compile nah setelah kalian compile akan ada file Start.exe di WP_Output kayak gini.

Kalian bisa rename Start.exe jadi setup.exe, lalu pindahin ke folder dimana ada setup-* nya, kayak gini.

Nah kalian tinggal buat folder Redist sesuai yang ada di script, selesai kalian punya installer CorePack sendiri, bisa kalian bagi ke adek, temen, tetangga, gue ga bertanggung jawab kalau kalian jual ini ya, gue cuman share doang wkwkwk.
